required files in docker container:

- *.iml with external annotations
- .idea/libraries -> no need, if using --rootfile /idea-project/pom.xml with ideainspect
- inspection profile xml file in .idea/inspectionProfiles/
- ~/.m2/repository with all deps : not really required but speeds up project opening since it avoids to download all deps again
